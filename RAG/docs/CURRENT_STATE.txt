# CURRENT STATE - RAG Training Project

Last Updated: 2025-12-01

## Project Status

### Overview
Multi-week RAG (Retrieval-Augmented Generation) training project implementing a production-grade arXiv Paper Curator system. The project progresses through 7 weeks, each building on previous functionality.

### Technology Stack

- **Backend:** FastAPI, Python 3.12+, UV package manager
- **Database:** PostgreSQL 16
- **Search:** OpenSearch 2.19 (hybrid search)
- **LLM:** Ollama (local) with llama3.2:3b-instruct-fp16
- **Workflow:** Apache Airflow 3.0
- **Embeddings:** Jina AI API
- **Monitoring:** Langfuse
- **Caching:** Redis
- **Agents:** LangGraph
- **Bot:** python-telegram-bot
- **Container:** Docker Compose
- **Development:** Ruff, MyPy, Pytest
